User Instructions:

In order to determine the appropriate workflow system to be used for book maintenance please answer the following questions:

1. Known Bookworm users please select "Skip Straight to BookWorm" tab.

2. For other systems select Source System from the drop down list, press the Next button.

3. Input Book Cost Centre or select Activity from drop down, press the Next button.

If you do not know your Book Cost Centre please contact your BUC representative.

4. You will be directed to the specific workflow system to be used.

For BookWorm and if your are a new BookWorm user please contact M&IBBookwormTraining&Communications@rbs.com to ensure you are correctly set up.

For SDM please note down the SDM ref, click Ok and you will be directed to SDM.

5. The Start Over button will clear inputted data, allow you to start again.
